# Counter Earth Dynamic Wallpaper Prompts

## How To Use

You have a **reference image** (the spring dawn scene) attached. Use it as the base for every edit. The prompts below tell the AI what to **change** — lighting, sky, and small details — while keeping the village layout, camera angle, art style, and composition identical.

**Workflow per season:**
1. Start with the season's dawn image as reference for the remaining 7 times of day
2. For other seasons: first generate that season's dawn (using spring dawn as reference + season change prompt), then use that as the reference for the rest of that season

## Style Reminder (keep consistent across all)

```
Maintain the stylised survival-adventure game art style: low-poly, matte textures,
clean readable shapes, handcrafted feel, warm natural tones, soft cinematic lighting,
atmospheric, Roblox-friendly. Keep 16:9 4K composition. No text, no UI, no characters.
```
